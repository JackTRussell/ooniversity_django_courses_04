Python&Django from PyBursa

Description

Курс Python/Django предназначен для начинающих программистов, которые уже имеют базовые знания Python и хотят освоить этот язык на профессиональном уровне для web-разработки.

Program

В этом курсе акцент сделан на изучение популярного web-фреймворка Django. Используя Django, вы сможете значительно ускорить процесс web-разработки, создавать мощные и высоконагрузочные web-приложения.

Perspectives

После обучения в этом курсе вы сможете легко претендовать на позицию "Junior web developer"
в ведущих IT-компаниях.

Program

First week: Основы Python

zen, pep8, pylint, tools, IDE, консоль, типы данных, операторы, синтаксис управляющих конструкций (if, for...), исключения и разбор, стектрейса, кодировка в Python

Second week: Строки и итераторы

работа со строками и итерируемыми типами, запуск скрипта и параметры, git http://try.github.com/levels/

Third week: Функции и модульность

функции (def, lambda), генераторы списков, модули, структура
