Курсы Python/Django от Pybursa

Description

Python — высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и читаемости кода. Синтаксис ядра Python минималистичен. В то же время стандартная библиотека включает большой объём полезных функций.

Program

Курс Python/Django предназначен для начинающих программистов, которые уже имеют базовые знания Python и хотят освоить этот язык на профессиональном уровне для web-разработки.
В этом курсе акцент сделан на изучение популярного web-фреймворка Django. Используя Django, вы сможете значительно ускорить процесс web-разработки, создавать мощные и высоконагрузочные web-приложения.

Perspectives

После обучения в этом курсе вы сможете легко претендовать на позицию "Junior web developer" в ведущих IT-компаниях.
